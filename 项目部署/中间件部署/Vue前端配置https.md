# Vue 前端配置 HTTPS

## 为什么前端必须用 HTTPS

WebRTC 协议强制要求页面在安全上下文中运行，即必须通过 `https://` 或 `localhost` 访问。由于项目部署在局域网内，需要通过 IP 访问（例如 `10.168.89.xxx`），因此必须为前端开发服务器启用 HTTPS，否则浏览器会阻止视频播放。

## 具体操作步骤

### 1. 生成自签名 SSL 证书

- 在 Vue 项目的根目录下新建一个隐藏文件夹（例如 `.cert`），用于存放证书文件。
- 使用系统自带的 `openssl` 工具生成自签名证书和私钥。
- 生成过程中，当提示输入 **Common Name** 时，务必填写你的**本机 IP 地址**（例如 `10.168.89.202`），否则浏览器仍会报安全错误。
